# projetFlask

## Installation

Pour installer le projet, suivez les étapes ci-dessous :

1. Clonez le dépôt :
  ```bash
  git clone
  https://github.com/OUSMANE-SACKO/projet-Graphql.git
  ```
2. Accédez au répertoire du projet :
  ```bash
  cd manga-backend
  ```
3. Activez l'environement virtuel venv:
  ```bash
  python3 -m venv venv
  source venv/bin/activate
  ```
